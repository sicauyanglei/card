# 证件照制作工具 - 设计文档

## 概述

纯前网页应用，用户通过相机自拍或相册选择照片，自动处理生成常用尺寸证件照，支持打印排版和单独下载。

## 核心功能

1. **照片输入**
   - 相机拍照（getUserMedia API）
   - 相册选择（File API）

2. **人像处理**
   - 人脸检测（MediaPipe Face Detection）
   - 自动裁剪（头部居中、上下比例标准化）
   - 背景替换（蓝底 #1E5AA8、白底 #FFFFFF）

3. **证件照生成**
   - 尺寸规格：
     - 一寸：25mm × 35mm（295 × 413 px @300dpi）
     - 二寸：35mm × 49mm（413 × 579 px @300dpi）
     - 小二寸：33mm × 48mm（390 × 567 px @300dpi）
   - 排版输出：A4 纸（可选择 6寸/5寸/4R 排版）
   - 单张输出：各尺寸 PNG 下载

4. **预览与下载**
   - 分页预览各尺寸
   - 单独下载
   - 打包下载（ZIP）

## 技术方案

- **纯前端**：照片处理全在浏览器本地完成
- **人脸检测**：MediaPipe Face Detection（Google 开源）
- **图像处理**：Canvas 2D API
- **抠图**：色差检测 + 边缘羽化
- **打包下载**：JSZip

## 界面流程

```
拍照/选择 → 人像检测 → 选择背景色 → 生成证件照 → 预览/下载
```

## 数据流

```
照片输入 → 人脸检测定位 → 人像抠图 → 背景填充 → 按尺寸裁剪 → 排版/单张输出
```

## 隐私说明

所有照片处理在用户本地设备完成，不上传到任何服务器。
