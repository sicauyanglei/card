# 证件照制作工具 - uni-app 版本

## 使用方法

### 方式一：HBuilderX 打开

1. 下载本项目到本地
2. 用 HBuilderX 打开 `uni-app` 文件夹
3. 在 HBuilderX 中运行：
   - **运行到 Chrome** - 网页预览
   - **运行到小程序模拟器** - 微信小程序
   - **运行到手机** - 真机调试

### 方式二：命令行

```bash
cd uni-app

# 安装依赖
npm install

# 开发模式
npm run dev:app

# 构建 App
npm run build:app
```

## 功能

- 拍照或从相册选择照片
- 自动人脸检测
- 多种背景色切换（蓝、白、绿）
- 生成多种尺寸证件照（一寸、二寸、小二寸）
- 打印排版下载

## 注意事项

1. 相机功能需要真机调试才能正常使用
2. 图片保存功能会请求相册权限
3. 高级图像处理功能（抠图、美颜）需要在 App 端实现

## 技术栈

- uni-app (Vue3)
- uni-app 内置 API
